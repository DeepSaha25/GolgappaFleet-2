# GolgappaFleet-2

## Project Info

Premium Golgappa Delivery Service.

## Getting Started

1.  Clone the repository
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```
